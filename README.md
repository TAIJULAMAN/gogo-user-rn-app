# GOGO User App

A feature-rich React Native application for on-demand delivery and logistics. Built with Expo, **GOGO** offers a seamless user experience for booking, tracking, and managing deliveries with a modern, premium design.

## üöÄ Features

### Core Functionality
-   **Smart Order Creation**: Intuitive multi-step wizard for selecting pickup/dropoff locations, choosing vehicle types, and secure checkout.
-   **Real-time Tracking**: Interactive map integration to visualize routes and track deliveries in real-time.
-   **Order Management**: View detailed order history, track status timelines (Placed -> Assigned -> Picked Up -> Delivered), and download professional PDF invoices.

### User Experience
-   **Smooth Animations**: Powered by `react-native-reanimated` for fluid transitions and engaging micro-interactions.
-   **Role-Based Architecture**: structured to support multiple user roles (User, Driver, Admin) with a clean separation of concerns.
-   **Secure Authentication**: robust onboarding and sign-in flows.

### Account & Utilities
-   **Address Book**: Save and manage frequently used addresses (Home, Office).
-   **Rating System**: Rate drivers and provide feedback after delivery.
-   **Privacy & Support**: Integrated privacy policy and support sections.

-   **Framework**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
